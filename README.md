# ✈️ Torino travel agency

**Tourino** is a travel and tourism web application where users can log in to their accounts, browse available tours, make reservations and payments, view transaction history and tour statuses, and update their personal information easily.

---


## 📌 Features

- ✅User authentication (Login)
- ✅Token stored in Cookies server side
- ✅Middleware redirect if isnt login
- ✅Theme dark/light mode
- ✅Filter tours with times & city name using query string
- ✅Reserve tour
- ✅Edit profile info
- ✅SSG/SSR/ISR/CSR pages
- ✅Responsive
- ✅Show status reserve tours
- ✅Show Bank name from cart number
---


## 🚀 Technologies Used

- **Next.js 14** –  The main framework used to build the React application. 
- **React 18** – The core UI library for building interactive user interfaces. 
- **Tailwind CSS** – A fast and modular utility-first CSS framework for styling.
- **React Query** – For managing API requests, caching, and server state. 
- **React Hook Form + Yup** – For efficient form handling and schema-based validation. 
- **Radix UI** – A set of accessible and fully customizable UI primitives. 
- **Lucide React** – A collection of clean and lightweight React SVG icons.  
- **Axios** – For making HTTP requests to APIs. 
- **Swiper** – For building responsive sliders and carousels.  
- **Sonner** – For displaying elegant toast notifications. 
- **Motion** – For modern, smooth animations in React.  
- **Dayjs Jalali / Zaman** – For handling and formatting Persian (Jalali) dates/persian date picker.  
- **Iran Bank Detector** – Detects the issuing Iranian bank based on card numbers.
- **Vanilla Tilt** - Adds 3D interactive tilt effects to elements.
- **Next Themes** - For managing dark/light mode and theme switching in Next.js.
